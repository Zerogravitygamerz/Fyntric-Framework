# Fyntric Framework

This repository contains scripts for Garry's Mod.

The `lua/autorun/client/cl_inventory.lua` script adds a simple inventory UI that toggles on and off when the player presses the Tab key. The UI covers the entire screen, blurring the game in the background with a subtle dark overlay. While the inventory is open, player movement and controls are disabled so only the UI can be interacted with.
