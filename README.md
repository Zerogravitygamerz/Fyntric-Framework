# Fyntric Framework

This repository contains scripts for Garry's Mod.

The `lua/autorun/client/cl_inventory.lua` script adds a simple inventory UI that toggles on and off when the player presses the Tab key. The UI covers the entire screen, blurring the game in the background with a subtle dark overlay. While the inventory is open, player movement and controls are disabled so only the UI can be interacted with. The centered panel shows two 5x5 grids of dark grey slots with action buttons in between for entering an amount, using, giving and closing the inventory.
